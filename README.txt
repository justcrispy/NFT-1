Bonjour,

Ce site est un site mirroir, il faut télécharger l'application pour suivre les étapes du site dans le logiciel.


Voici le site qui est disponible dans le fichier Accueil.html

- Pour retourner à l'accueil : Appuyer sur "NFT FACTORY"

- Aller voir les exemples NFT : Appuyer sur "Galerie"

- Voir le fonctionnement du site : Appuyer sur "Comment ça fonctionne ?"

- Voir les personnes qui ont contribués au projet : Appyuyer sur "Qui nous-sommes ?"

- Commencer la génération des images NFT : Appuyer sur "Commencer"


Etape 1 : Choisir l'arrière plan de l'image NFT.
Etape 2 : Choisir la forme de visage deu personnage.
Etape 3 : Choisir la couleur de peau du personnage.
Etape 4 : Télécharger le logiciel pour générer les images. 

Une fois téléchargé, suivre les étapes ci-dessous (1 à 3) avec le format du site en aide.

Lorsque vous aurez téléchargé, vous devrez le décompresser.
Une fois le dossier Projet décompresser, rendez-vous dans le dossier "dist", puis "nft1". Dans se dossier, se trouve l'executable "nft1.exe".
Lorsque vous le lancerez, il va vous demander où se trouve le chemin d'accès du dossier "Projet". 
Une fois le chemin reseigné, il va vous demander de choisir un background, c'est à dire le fond de l'image. Il y a 9 couleurs différentes : beige; bleu; cyan; marron; orange; rose; rouge; vert et violet.
Lorsque vous aurez choisi votre background, vous devrez choisir la forme de votre visage : carré ou rond.
Ensuite, la couleur de peau qui vont de 1 à 5 avec 1 étant le plus clair et 5 le plus foncé.
Et pour finir, le nombre de NFT que vous voulez générer.

Tapez sur "Entrée" lorsque vous aurez finis de rentrer chaque information

Exemple :

Indiquer dans quel répertoire se trouve le projet ? : C:/Users/quent/OneDrive/Bureau
indiquer la couleur du background souhaité : bleu
indiquer la forme du visage (carré ou rond) : rond
indiquer la couleur du visage : 3
Combien de NFT voulez-vous générer ? : 5
